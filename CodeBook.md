#Code Book

This code book summarizes the resulting data fields in 'tidy.txt'.
Tidy dataset 'all_data' which is created by 'run_analysis.R' script and contains average of each variable for each activity and each subject based on the dataset.

##Data source:

This dataset is from the "Human Activity Recognition Using Smartphones Data Set" which is avaiable here: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

##Identifiers

- 'subject' - The ID of the test subject
- 'activity' - The type of activity performed when the corresponding measurements were taken
